## 📦 Getting Started

To run any project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mehdix17/Web-Dev-Projects.git
   cd Web-Dev-Projects
   ```

2. **Navigate into a project folder**
   ```bash
   cd Weather-App   # or any other project
   ```

3. **Follow the project-specific setup** (install dependencies if needed, then open `index.html` or start the server)

> Each project is self-contained in its own folder. For projects with a Python backend, make sure to install dependencies with `pip install -r requirements.txt` before running.
